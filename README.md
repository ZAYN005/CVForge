CVForge is a full-stack web application designed to help users create professional resumes quickly and efficiently. The system collects user information step-by-step, securely stores data in a database, and generates customizable CVs instantly using professional templates.

This project demonstrates real-world full-stack development including authentication systems, admin dashboards, data analytics, and document generation.

Core Features

User Features
- Step-by-step CV data collection workflow
- Multiple professional CV templates
- Instant CV generation
- Export CV as PDF
- Secure email-based user authentication
- Save and manage previous CV versions
- Edit or delete saved CVs anytime
- User profile dashboard
- CV creation history tracking

---

- User profile analytics dashboard
- Track number of CVs generated
- Monitor user activity
- Data usage insights

- Secure admin login panel
- Monitor total registered users
- Track user activity statistics

System Workflow

1. User registers using email authentication
2. System collects CV data step-by-step
3. Data is stored securely in MongoDB
4. User selects preferred CV template
5. CV is generated instantly
6. User can export CV as PDF
7. User dashboard tracks CV history
8. Admin panel monitors system usage
